# Password Strength Tester 🛡️
A lightweight and efficient tool designed to evaluate the robustness of passwords. This project analyzes security entropy and identifies common vulnerabilities to help users create uncrackable credentials.
## 🚀 Features
+ <b>Multi-Factor Analysis:</b> Checks for uppercase, lowercase, numbers, and special characters.
+ <b>Entropy Scoring:</b> Calculates the mathematical strength based on character diversity and length.
+ <b>Real-time Feedback:</b> Provides instant security ratings.
+ <b>Minimalist Design:</b> Clean code architecture focused on performance and security.

```Bash

```
2. Navigate to the directory:
```Bash
cd Password_strength_tester
```


> TIP: You can also run this program with the provided `.exe` file easily

## 🖥️ Usage
Run the program and enter a password to see its strength evaluation


# 📊 Strength 
| Level | Description |
| :--- | :--- |
| <kbd>Weak</kbd> | Short length or only one character type. |
| <kbd>Medium</kbd> | Mix of letters and numbers; moderate length. |
| <kbd>Strong</kbd> | Complex mix including symbols; high entropy. |
